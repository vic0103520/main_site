An Introduction to Modal Logic

Instructor: Churn-Jung Liau

This lecture covers basic modal logic and its applications (in particular, in AI and distributed systems). Specific topics include: Kripke semantics, normal modal systems (K, D, S5, etc.), completeness theorem, finite model property, decidability, multi-agent epistemic logic, dynamic logic.

Prerequisites

Elementary Logic

Syllabus

  • basic modal logic: syntax, Kripke semantics, model-theoretic notions
  • normal systems of modal logic: K, D, T, S4, S5, etc.
  • meta-theorems of normal systems: soundness, completeness, decidability
  • generalization of basic modal logic: multi-dimensional and multimodal logic
  • multi-agent epistemic logic
  • dynamic logic
