宗旨

近年來，從事基礎計算科學研究之台灣學者們已在各校各自成立研究團隊並相互合作。對基礎計算科學有興趣之學生須透過一系列課程學習基本知識；然而，國內從事此類研究的學者分散在各校，難以僅靠一己之力開設整套課程。因此，我們邀請有相同興趣的學者一同開課，訓練下一代的研究人才。

「邏輯、語言與計算」暑期研習營希望培養學員獨立進行基礎計算科學研究之能力。從2008年起，本研習營在兩大主題之間輪流替換。自 2012 年起，本研習營正式成為台灣大學暑修課程，針對大學部學生開課，但亦歡迎研究生選修。台灣大學學生可透過國立台灣大學暑期課程網選課，有學籍之其他學校學生可透過校際選修選課。

「邏輯、語言與計算」暑期研習營
Formosan Summer School on Logic, Language, and Computation

今年主題：型別原理

程式語言學的研究對象是程式，可以想成是「寫得很詳細、可以一步步照著做的規則／流程」；程式不一定非得和電腦扯上關係，像 SOP（標準作業流程）和飛行員必用的 checklists（檢查表）就是寫給人執行（照著做）的程式。程式語言學的最大特點是用符號作為主要工具，精準（無歧義）簡練（不冗長）地表示出程式和相關概念，以及研究它們的各種性質。除了精準簡練以外，我們喜歡的符號有以下這些特質

  • 厲害：用符號把關於程式的事情好好寫下來之後，我們就可以用數學方法確鑿地談這些事情，用數學證明確保厲害的好性質，例如「符合某些條件的程式絕對不會出錯」。
  • 易用：符號是我們解決程式相關問題的工具，在符號設計上我們會有所講究，為各種問題量身設計順手易用的專門符號。
  • 內涵：好的符號往往有多個面向層次的意義，幫助我們用不同角度思考問題；
我們特別喜歡向數學和邏輯學借用符號，和他們長久累積發展的深刻概念互通。

Session Types

2018 年邀請到英國倫敦帝國學院 Imperial College London 的吉田展子教授和 Rumyana Neykova 研究員為我們講授 π-calculus 與 session types。吉田教授長年致力於並行計算模型與型別系統的研究，不僅學術成果豐碩，亦與業界密切合作，是介紹此領域理論與實踐的不二人選。Neykova 研究員以他在 “multiparty session types” 上的研究取得博士學位，除了設計理論框架，更將 multiparty session types 實作於 Python, Erlang 等主流語言，並累積豐富的教學經驗。

2019 年邀請了 Jun Furuse 以 Tezos 為案例來為我們講解 formal verification 在業界的應用。Furuse 先生會由淺入深逐步介紹：blockchain、smart contract、為何業界需要 formal verification，以及，業界如何上述這些領域範圍中應用 formal verification。

適合幾年級的學生呢？
根據我們的統計，歷年來參加者的年齡分布很廣，除了大學生與研究生外，還有少數高中生跟不少的社會人士。課程沒有預設背景，不同年級的學員（盡力參與）都可以跟得上課程，我們也有助教群可以協助學員。
課程很難嗎？

因為各種科系背景的不同，每個人狀況都不大一樣。大致上我們發現科系背景間的差異較大，例如歷年來數學系跟資工系高分通過的比較多，但也曾有財金系的同學拿到滿分哦！至於不及格的同學則是沒有明顯的差異，即便是最相關的資工系也有不少人最後沒拿到學分（殘念⋯⋯）
