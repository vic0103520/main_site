邀請演講 I / Invited Talk I

  • 講者: 唐鳳 Audrey Tang
  • 講題: 函數程式設計的商業應用 / Commercial Uses of Functional Programming
  • 簡報 (線上瀏覽): http://www.slideshare.net/autang/commercial-uses-of-functional-programming
  • 簡報 (PDF下載): http://audreyt.org/tmp/FLOLAC14.pdf
  • 錄音: https://soundcloud.com/audrey-tang/cufp-flolac14
  • 逐字稿共筆 (待補完): https://hackpad.com/CUFP-FLOLAC14-PoPV1V9wVse

Functional programming has been at the forefront of a new generation of programming technologies: Companies employing functional programming use it to enable more effective, robust, and flexible software development. This talk gives a brief overview of the growing community of commercial users of functional programming, as well as personal anecdotes using Haskell in commercial settings.

函數程式設計一直引領著新一代的編程科技：接納函數程式設計的公司借其力量讓軟體開發成效更佳、穩健又富有彈性。這次演講將簡短介紹日益增長的函數程式設計商業用戶社群之概況，以及個人於商業環境使用 Haskell 的軼事。
